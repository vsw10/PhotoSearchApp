# PhotoSearch
 A photo search Application to use Flickr Search APIs to display images as per the entered query

v1.0 (Thursday May 6th,2021) PhotoSearch 
package="com.accion.photo"
This App is created with Java Language.

Frameworks and Libraries being used are mentioned as under:
1. MVVM
2. Live Data
3. DataBinding
4. Volley
5. Retrofit (Testing needs to be done)
6. Glide

ConstraintLayout and Recycler Views have been used.

Added Features:
Added Debug and Error Logs
Documentation for all methods and API calls
 

Following Design Patterns being used:
1. Singleton Design Pattern
2. Factory Design Pattern
3. Observer
4. Aggregation
5. Adapter Pattern
